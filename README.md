# LlanquihueTourApp

## Descripcion

Este proyecto es una aplicacion en Java para la gestion de guias turisticos de la agencia Llanquihue Tour.

Permite leer datos desde un archivo CSV, guardarlos en una lista, mostrarlos por consola y hacer busquedas simples por nombre.

Tambien se aplican conceptos de programacion orientada a objetos como encapsulamiento y composicion.

## Paquetes

* app: contiene la clase Main
* model: contiene las clases Persona, Guia y RutaTuristica
* service: contiene la logica de busqueda y muestra de datos
* util: contiene la lectura del archivo CSV

## Como ejecutar

1. Abrir el proyecto en IntelliJ IDEA
2. Verificar que el archivo guias.csv este en la raiz del proyecto
3. Ejecutar la clase Main
4. Ver los datos en consola
